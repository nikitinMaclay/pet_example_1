# Nebula Studio — пример многостраничного сайта

Демонстрационный сайт веб-студии, созданный как пример работ. Чистый HTML/CSS/JS
без сборщиков и зависимостей — открывается в любом браузере и легко публикуется.

## Страницы

| Файл | Описание |
|------|----------|
| `index.html` | Главная: герой, статистика, услуги, избранные работы |
| `about.html` | О студии: подход, таймлайн, команда |
| `services.html` | Услуги, тарифы и процесс работы |
| `portfolio.html` | Портфолио с фильтрами по категориям |
| `contact.html` | Контакты и форма обратной связи (демо) |

## Особенности

- 📱 Адаптивная вёрстка (мобильное меню, гибкие сетки)
- 🎨 Современный тёмный дизайн с градиентами
- ✨ Анимации появления при скролле и счётчики
- 🔎 Фильтр проектов в портфолио
- ⚡ Без зависимостей и сборки

## Запуск локально

Просто откройте `index.html` в браузере. Либо запустите локальный сервер:

```bash
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

## Публикация через GitHub Pages

1. Запушьте репозиторий на GitHub.
2. Settings → Pages → Source: ветка с сайтом, папка `/ (root)`.
3. Сохраните — сайт будет доступен по адресу
   `https://<username>.github.io/<repo>/`.
